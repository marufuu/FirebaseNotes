# Firebase Notes & Media Review App

**Personal Project** | iOS (SwiftUI/UIKit, Firebase)

---

## 📱 What This App Does

An iOS app that lets users:
- ✅ Sign in with Email/Password or Google account
- ✅ Create and manage personal notes
- ✅ Review movies and books with ratings
- ✅ Save favorites and like content
- ✅ Sync everything in real-time

---

## 🛠 What I Built

| Feature | What I Did |
|---------|------------|
| **Authentication** | Email/Password + Google Sign-In with user profiles |
| **Notes** | Create, update, delete personal notes |
| **Media Reviews** | Rate movies/books, like, and favorite system |
| **Backend** | Firebase for auth, storage, and real-time sync |

---

## 🔧 Tech Stack

- **Language:** Swift 5.9
- **UI:** SwiftUI / UIKit
- **Backend:** Firebase (Auth, Firestore, Real-time sync)
- **Minimum iOS:** 15.0

---

## 🚀 How to Run

1. Clone the repo
2. Add `GoogleService-Info.plist` (from Firebase Console)
3. Open `FirebaseNote.xcodeproj` in Xcode
4. Press `⌘ + R` to build and run
